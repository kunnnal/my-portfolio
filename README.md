# DevOps Command Center Portfolio

High-end personal portfolio built with Next.js App Router, TypeScript, Tailwind CSS, GSAP, Lenis, and React Three Fiber.

## Run locally

```bash
npm install
npm run dev
```

## Stack

- Next.js App Router
- TypeScript
- Tailwind CSS
- GSAP
- Lenis
- Three.js via React Three Fiber

## Customization points

- Update personal content in `lib/portfolio-data.ts`
- Adjust theme tokens in `lib/theme.ts` and `app/globals.css`
- Refine the hero scene in `components/canvas/hero-network-canvas.tsx`
